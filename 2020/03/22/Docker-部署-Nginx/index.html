3.创建Nginx容器

  • /home/nginx/conf/nginx.conf 为刚才创建配置文件的路径
  • /home/nginx/logs 日志保存的位置
  • /home/nginx/www 跳转静态资源位置

4.其他命令

  • docker ps | grep nginx 查看nginx容器运行状态
  • docker start mynginx 启动
  • docker start mynginx 重启
  • docker stop mynginx 暂停
  • docker rm mynginx 移除
